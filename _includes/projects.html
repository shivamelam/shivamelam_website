Projects

Local Registration System – A PC2(Programming Contest Control) Extension

  • A web application that uses RESTful web services to register contestants for ACM’s ICPC at local Universities
  • Stores the data in a local database which can be accessed by PC2 (Programming Contest Control) system
  • The information from PC2 is pushed into Baylor Contest Management System
  • Technologies Involved: Dropwizard, AngularJs, Bootstrap, Java, MySQL, Liquibase, Hibernate, RAML

Crypto Alert

  • A Desktop application that lets users to configure alerts for various crypto currencies
  • Provides information about the current price of each currency which refreshes for every 10 seconds
  • User can input a target price for available crypto currencies beyond which a desktop notification appears
  • Technologies Involved: Electron, Angular5, HTML, CSS, Node.js

My Mobile Tracker

  • An application developed for Android based mobile users that helps to keep track of data usage

  • Technologies Involved: Java, Android SDK, .NET, SQL Server, JSON
